## Learning goals

After this course, student is able to:

- Test software with common software testing methods
    - Unit tests
    - Integration tests
    - Smoke test
- Automate software tests
- Do code coverage and test coverage
- Knows logging best practices
- Can implement a simple CI/CD pipeline
